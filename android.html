Android (grekshe andro — „inson, er adam“, tiyisli qosımshası — „robot“; „odamsimon robot“) — smartfonlar, planshetler, elektron kitaplar, cifrlı pleyerlar, qol saatları, fitnes bilezikler, oyın pristavkalari, noutbuklar, netbuklar, smartbuklar, Google Glass kóz áynekleri[2], televizorlar, proyektorlar hám de taǵı basqa qurılmalar (2015-jılda avtomobil ko'ngilochar sistemaları hám xojalıq robotlarga da ornatildi) ushın operatsion sistema esaplanadı. Daslep bul operatsion sistema Android Inc. tárepinen islep shıǵılǵan, keyinirek onı Google satıp aldı. Operatsion sistema Linux yadrosı hám Google tárepinen islep shıǵılǵan Java virtual mashinasına tiykarlanǵan. Keyinirek, Google bul platformani qollap-quwatlaw jáne onı jáne de rawajlandırıw menen shuǵıllanatuǵın Open Handset Alliance (OHA) sho'basini jarattı.

Android operatsion sisteması qurılmanı Google tárepinen islep shıǵılǵan kitapxanaları arqalı basqarılatuǵın Java qosımshaların jumısqa túsiriw imkaniyatın beredi. Android Native Development Kit sizge C hám basqa programmalastırıw tillerinde jazılǵan kitapxanalar hám programma komponentleri menen islew imkaniyatın beredi. 2014-jıldıń ekinshi shereginde dúnya boylap sotilgan smartfonlardıń 86 %'da Android sisteması ornatılǵanlıǵı hám de 2017-jıl may ayında islep shıǵıwshılar konferenciyasında Google pútkil Android tariyxında 2 milliarddan artıq Android qurılmaları aktivlashtirilganligini járiyaladı.
